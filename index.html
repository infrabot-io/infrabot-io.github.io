infrabot - is a tool which gives a possibility to administer infrastructure using a Telegram Bot

Getting Started Download

Donation

Infrabot is on GitHub and licensed under an AGPL-3.0 license, which basically means that you are free to get and use it for commercial and non-commercial purposes subject to some simple license conditions, however you can donate to support author for a cup of beer.

Control your infrastructure from Telegram. controllability

You can easily control your infrastructure via your Telegram Bot. Send commands to Bot for execution when you are unable to get to the computer to fix problems when they happen.

Advanced configuration possibilities. configuratibility

There are many configuration possibilities of the Bot which can be done via Config editor. All parameters are well named and structured, however you can find detailed information about each of them on the documentation page.

Plugin system support. flexibility

You can easily create\edit and configure your plugins using preinstalled Plugin editor tool.
